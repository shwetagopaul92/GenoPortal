Different file formats
  1. BED :

    A BED file (.bed) is a tab-delimited text file that defines a feature track.BED (Browser Extensible Data) format provides a flexible way to define the data lines that are displayed in an annotation track. BED lines have three required fields and nine additional optional fields. The number of fields per line must be consistent throughout any single set of data in an annotation track. The order of the optional fields is binding: lower-numbered fields must always be populated if higher-numbered fields are used.

  2. VCF :

    VCF is a text file format (most likely stored in a compressed manner). It contains meta-information lines, a header line, and then data lines each containing information about a position in the genome.

  3. GTF :

    The Gene transfer format is a file format used to hold information about gene structure. It is a tab-delimited text format based on the general feature format, but contains some additional conventions specific to gene information.
